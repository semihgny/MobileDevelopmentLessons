# Mobile Development Lessons

This repository contains a collection of mobile application projects and assignments completed during our **Mobile Development** university course. Each folder represents a different topic or mini-project we covered in class, demonstrating various concepts and capabilities of Android mobile application development.

## Course Projects & Topics Covered

This repository serves as an educational portfolio showcasing the hands-on projects we developed throughout the semester. Below are the key concepts and applications explored:

- **Core Android Components**: 
  - Intent Usage (`Intent Kullanımı`): Navigating between activities and passing data.
  - Menu Design (`Menu Tasarimi`): Creating options and context menus.
  - ListView Usage (`ListviewKullanimi`): Displaying lists of data interactively.
- **Data Storage & Persistence**: 
  - Local Database (`KitapVeritabani`): Storing book records using SQLite.
  - SharedPreferences (`Tercihler`): Saving user preferences and simple key-value data locally.
  - File I/O (`Metin Dosyası`): Reading and writing data to text files on the device.
- **Hardware & Media Integration**: 
  - Sensor Usage (`SensorKullanimi`): Accessing device sensors like accelerometers or proximity sensors.
  - Media Player (`Mediaplayer`): Playing audio/video within the app.
- **Full-Stack Mobile Applications (Node.js & MySQL)**:
  - Real-time Chat App (`Chat_MySql_Nodejs`): A chat application utilizing a custom Node.js/MySQL backend.
  - Book Registry System (`KitapKayit_MySQL_Nodejs`): A remote database application storing records via a custom backend API.
- **Mini Applications**:
  - Calculator (`Hesap Makinesi`): A fully functional calculator app.

## Purpose

The primary goal of this repository is to archive our mobile coursework and provide practical code examples for students or anyone learning Android App Development. These examples range from basic UI components to full-stack integration with external APIs and databases.

## Requirements

- Android Studio (Recommended)
- Java / Kotlin (depending on the specific project configuration)
- Node.js & MySQL (Required only for the specific backend integrated projects like `Chat_MySql_Nodejs`)

## Usage

Clone the repository and open any specific project folder in Android Studio:
